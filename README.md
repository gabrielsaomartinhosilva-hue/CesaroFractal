# Automated Design and Miniaturization of Cesàro Fractal-Based RF Resonators for Chipless RFID Tags

## Submission ID: 10722

**"Automated Design and Miniaturization of Cesàro Fractal-Based RF Resonators for Chipless RFID Tags"**
